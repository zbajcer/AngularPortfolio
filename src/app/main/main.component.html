Web development:

Since December of 2018 I work in Generali Osiguranje d.d. as web developer. I maintain and develop existing insurence application. Programming is also my favourite hobby. In free time I developed this great application in Angular framework. It is great because of backend which is developed in Java SpringBoot. It is also connected to MySQL database. In the section "Currency converter" you can choose currency list by date and my app is dinamicaly checking if the choosen currency by date exists in database. If doesnt exist, it connects to HNB web api, downloads currency in JSON, store values in database and proceeds with serving those currencyes to frontend, Angular. I have also implemented statistics. On every click it updates counter of each chosen start value.

I have also developed some simple apps which you can find on my GitHub account.
